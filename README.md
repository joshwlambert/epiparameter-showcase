# Epiverse-TRACE {epiparameter} Tool Deep dive

This repository contains the presentation for the Epiverse-TRACE tool deep dive to be presented at the Epiverse _Strengthening Adoption of Epiverse Tools in Africa_ Workshop, Diamniadio, Senegal, December 2025.

The quarto presentation is `epiparameter-showcase.qmd`, the HTML version is `epiparameter-showcase.html`.

The images in the presentation are in `images/`. 

The quarto presentation uses a custom theme: `es-theme.scss`.
